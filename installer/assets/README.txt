ISX Daily Reports Scrapper
==========================

A powerful tool for downloading and processing daily reports from the Iraqi Stock Exchange (ISX).

Features:
- Download daily reports from ISX website
- Process Excel files into organized CSV reports  
- Extract market indices and statistics
- Web-based interface for easy operation
- Licensed software with professional support

Installation:
1. Run the installer executable
2. Follow the installation wizard
3. A desktop shortcut will be created automatically
4. Launch the application from the desktop shortcut

Usage:
1. Double-click the desktop shortcut to launch the web interface
2. Set your desired date range for data collection
3. Click "Start Scraping" to download and process ISX data
4. View results in the web interface

System Requirements:
- Windows 10 or later
- Internet connection for downloading ISX data
- 100MB free disk space
