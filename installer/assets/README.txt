ISX Daily Reports Scraper - Installation Guide

Welcome to the ISX Daily Reports Scraper installation wizard!

WHAT IS ISX DAILY REPORTS SCRAPER?
==================================
ISX Daily Reports Scraper is a comprehensive web scraping and analytics platform for the Iraqi Stock Exchange (ISX). It automatically downloads, processes, and visualizes daily trading reports from the ISX portal.

FEATURES:
- Automated daily report downloading from ISX portal
- Excel to CSV data processing with forward-fill capabilities
- Modern web dashboard with real-time updates
- Professional licensing system with Google Sheets integration
- Individual ticker analysis with candlestick charts
- Market index extraction (ISX60, ISX15)
- Comprehensive data analytics and visualization

SYSTEM REQUIREMENTS:
===================
- Windows 7 SP1 or later (64-bit)
- 4 GB RAM minimum (8 GB recommended)
- 2 GB free disk space
- Internet connection for data scraping and license activation
- Chrome or Chromium browser (for web scraping)
- Microsoft Visual C++ Redistributable 2015-2022 (will be installed if needed)

INSTALLATION OPTIONS:
====================
During installation, you can choose:
- Desktop icon creation
- Start menu shortcuts
- File associations for .xlsx files
- Add to Windows PATH variable
- Auto-start with Windows
- Windows Firewall configuration

AFTER INSTALLATION:
===================
1. The installer will create shortcuts in your Start Menu
2. Launch "ISX Daily Reports Scraper Web Interface" to start using the application
3. Configure your license by running "Configure License" from the Start Menu
4. Set up Google Sheets integration for license management (optional)
5. Begin scraping ISX data through the web interface

IMPORTANT NOTES:
===============
- This is a LICENSED software. You will need a valid license key to use the application
- The application requires administrator privileges for initial setup
- Windows Firewall rules will be configured to allow the web interface (port 8080)
- The first run will prompt you to enter your license key
